# 虚拟鼓游戏 (AirDrum 3D Pro - MVP)

## 项目概述

虚拟鼓游戏是一款基于手势识别的音乐节奏游戏，结合MediaPipe手部追踪和RGB-D深度摄像头，提供沉浸式的交互体验。玩家通过挥动双手和进行捏合手势来击打虚拟鼓并控制音量。

## 核心功能

### 1. 难度选择系统 (Difficulty Selection)
- **三个难度等级**：简单、普通、困难
- **难度差异**：
  - 简单 (Easy)：音符下降速度慢，生成密度低
  - 普通 (Normal)：标准游戏体验
  - 困难 (Hard)：音符下降速度快，生成密度高
- **触发方式**：点击 "Game Mode" 按钮时弹出难度选择对话框
- **位置**：对话框显示在屏幕中央，支持点击或手势选择

### 2. 游戏模式控制

#### 暂停/继续 (Pause/Resume)
- **快捷键**：按空格键暂停/继续游戏
- **功能**：
  - 暂停时停止音符下降和音乐播放
  - 显示暂停菜单提示
  - 支持恢复游戏或按ESC退出

#### 游戏内音效 (Note Hit Sound)
- **音效反馈**：击中音符时播放打击音效
- **分类反馈**：
  - Perfect 击中：鼓声 + 钢琴音（强化反馈）
  - Good 击中：鼓声（基础反馈）
- **增强体验**：视觉+听觉双重反馈，提升游戏节奏感

### 3. 音符系统

#### 短音符 (Short Notes) - 点击式
- **外观**：圆形，带白色边框
- **触发方式**：敲击（击打手势）
- **判定方式**：点击时进行判定
- **生成比例**：50% 概率

#### 长音符 (Long Notes) - 按住式
- **外观**：长条形，显示起点和终点
- **触发方式**：捏合（拇指+食指）
- **持续时间**：随机 1500-2500 毫秒
- **判定方式**：需要在长条通过判定线期间始终保持捏合状态
- **视觉反馈**：
  - 未按住：原色长条
  - 按住中：绿色长条
- **生成比例**：50% 概率
- **同轨防重**：同一轨道的短音符和长音符不会重叠

### 4. 手势交互

#### 长按手势 (Long Press Gesture)
- **使用指尖**：拇指(Thumb) 和 食指(Index)
- **检测方式**：计算两指尖欧氏距离
- **捏合阈值**：约 0.05（归一化）
- **应用场景**：控制长条音符

#### 音量控制 (Volume Slider)
- **使用指尖**：拇指(Thumb) 和 小指(Pinky)
- **交互方式**：在屏幕右侧上下滑动调节
- **音量范围**：0%-100%
- **独立检测**：与长条音符的捏合手势分离，避免冲突

## 游戏玩法

### 游戏流程
1. **启动游戏**：运行程序进入主界面
2. **选择难度**：点击 "Game Mode: OFF" 按钮
   - 弹出难度选择对话框（屏幕中央）
   - 选择简单、普通或困难
3. **开始游戏**：难度选择后自动进入游戏
4. **击打音符**：
   - 圆形音符：敲击鼓位置
   - 长条音符：按住并维持捏合直到结束
5. **控制音量**：拇指+小指捏合，上下移动调节
6. **暂停游戏**：按空格键暂停，再按继续或ESC退出

### 判定系统
- **Perfect (完美)**：在判定范围内击打短音符或全程按住长音符
  - 得分：100分
  - 反馈：视觉+音效（鼓声+钢琴音）
- **Good (良好)**：在判定范围外稍微偏离击打短音符
  - 得分：50分
  - 反馈：视觉+音效（鼓声）
- **Miss (失误)**：错过或未完成
  - 得分：0分，连击归零
  - 反馈：视觉提示"Miss"

## 技术架构

### 核心技术栈
- **Python 3.9+**
- **PyQt5**：UI框架
- **OpenCV (cv2)**：图像处理和显示
- **MediaPipe**：手部追踪和姿态检测
- **Pygame**：音频播放和混音
- **NumPy**：数值计算

### 关键模块
- **MediaPipe Hands**：实时手部关键点检测（21个关键点）
- **RGB-D 摄像头**（Orbbec）：深度信息获取
- **音频合成**：钢琴弦音算法生成
- **粒子系统**：击打特效渲染

```
    4    8
    │    │

 │              │
11──12──13────14──15
       │
       16
```

- **4**: 拇指指尖 (Thumb Tip) → 长条音符 + 音量控制
- **8**: 食指指尖 (Index Tip) → 长条音符
- **20**: 小指指尖 (Pinky Tip) → 音量控制

## 使用说明

### 启动游戏
```bash
python main.py
```

### 游戏界面元素
- **Start**：启动摄像头
- **Auto Calibrate**：自动校准深度中心
- **Sensitivity**：调节深度检测灵敏度
- **Volume**：调节整体音量
- **Debug Mode**：显示深度图和调试信息
- **Hand Skeleton**：显示手部骨骼追踪
- **Mirror**：镜像显示摄像头画面
- **Game Mode**：启动/退出游戏模式

## 游戏分数计算

- 每个 Perfect：+100分
- 每个 Good：+50分
- 每个 Miss：0分，连击重置
- 连击倍数：最多显示连击数

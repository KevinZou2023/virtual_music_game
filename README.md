# AirDrum 3D Pro - 虚拟打击乐系统

**AirDrum 3D Pro** 是一款基于计算机视觉的虚拟打击乐应用，允许用户通过隔空手势演奏架子鼓和钢琴。无需昂贵的传感器，仅需普通摄像头即可体验（支持 MediaPipe 手部追踪），同时也支持专业的 Orbbec 3D 深度相机。

## ✨ 核心特性

- **虚拟架子鼓**: 包含 Kick, Boom, Tom 等鼓组，通过空间交互触发。
- **虚拟钢琴**: 支持手势或键盘触发的合成钢琴音色。
- **双重追踪模式**:
    - **手部骨骼模式 (推荐)**: 使用 MediaPipe 追踪手部关节，无需特殊硬件，普通摄像头即可，延迟极低。
    - **深度图模式**: 专为 Orbbec 3D 相机设计，基于真实深度数据判断。
- **视觉反馈**: 炫酷的粒子特效、动态 UI 和实时深度反馈。
- **自定义设置**: 支持灵敏度调节、自动校准和音量控制。

## 📋 系统要求

- **操作系统**: Windows 10/11
- **Python 版本**: 3.9+
- **硬件**: 普通网络摄像头 或 Orbbec RGB-D 摄像头

## 🎮 操作指南

### 界面控制

| 控件 | 说明 |
| :--- | :--- |
| **Start / Stop** | 开启或暂停摄像头与追踪 |
| **Hand Skeleton** | **勾选**启用普通摄像头模式 (MediaPipe)<br>**取消勾选**启用深度相机模式 |
| **Auto Calibrate** | 自动检测背景深度并校准触发平面 |
| **Sensitivity** | 调整触发灵敏度 |
| **Volume** | 调整全局音量 |

### 演奏方式

- **空气架子鼓**: 在摄像头前挥动双手，击打屏幕上的虚拟鼓面。
- **手势音量**: 拇指与食指捏合 (Pinch) 可控制音量推杆。
- **键盘辅助**:
    - `A`, `S`, `D` -> 触发鼓声
    - `Z` ~ `M` -> 触发钢琴音符
    - `Q` -> 退出程序

## 🛠️ 常见问题 (FAQ)

- **Q: 启动后画面黑屏？**
  - A: 摄像头正在初始化，请等待几秒。如果长时间无反应，请检查摄像头权限或是否被其他程序占用。

- **Q: 手部追踪不灵敏？**
  - A: 请确保环境光线充足，手部完整出现在画面中。尝试点击 "Auto Calibrate" 重新校准。

- **Q: 想要自定义鼓声？**
  - A: 将你的 `.wav` 音频文件放入 `assets/` 目录，并修改 `main.py` 中的加载逻辑。
